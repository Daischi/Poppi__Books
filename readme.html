📚 Poppi Book - Seu Portal de Livros Responsivo 📚

Bem-vindo ao Poppi Book, o projeto de um site de livros feito para treinar e demonstrar responsividade para todos os tipos de tela. Navegue, descubra e desfrute de uma experiência de leitura perfeita em qualquer dispositivo!

🌟 Objetivo

O objetivo deste projeto é demonstrar a capacidade de criar sites responsivos que se adaptam automaticamente às dimensões da tela do dispositivo, proporcionando uma interface intuitiva e fácil de navegar em qualquer plataforma. Ao projetar o AluraBooks, nos concentramos em oferecer uma experiência visual agradável, acessibilidade universal e funcionalidades práticas, tudo isso mantendo a essência da leitura digital.

🛠️ Tecnologias Utilizadas

  • HTML5: Estrutura semântica e organizada.
  • CSS3: Estilos elegantes com foco em responsividade.
  • JavaScript: Interatividade e funcionalidades dinâmicas.
  • Media Queries: Para adaptar o layout a diferentes tamanhos de tela.
  • Bootstrap: Biblioteca de componentes front-end para construir rapidamente layouts responsivos e personalizáveis.
